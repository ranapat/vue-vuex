# new-project

To start:

```bash
$ npm install
```

To develop:

```bash
$ npm run dev
```

To build for production:

```bash
$ npm run build
```

To lint you code:

```bash
$ npm run lint
```

To run unit tests

```bash
$ npm run unit
```

To run unit tests with --watch

```bash
$ npm run tdd
```

To run unit tests and e2d

```bash
$ npm run test
```

Docsify init

```bash
$ npm run docs-init
```

Docsify serve

```bash
$ npm run docs-serve
```
